## 特性

1. 《流畅的Python》的jupyter notebook实现和一些笔记，包括**所有代码**的注释。
2. 包含**所有章节**的重点资料，包括本章小结和延伸阅读等。
3. 每一个小结开头都有**TLDR**（Too Long, Don't Read，太长不看版），可以**速览本节内容**。
